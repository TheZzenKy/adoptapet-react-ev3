1. Componente: Lo uso para dividir la interfaz. Es el adecuado ya que hace que el código se vea mas ordenado.
2. JSX: En la interfaz también. Es el adecuado por que es el lenguaje que usa react y describe como se vería la pantalla (parecido a un HTML)
3: Props: Se usa como datos entre los componentes. Es el adecuado por que asi los componentes no dependen de datos fijos.
4. Estado: UseState: Se usa en el app.jsx. Es el adecuado por que cuando un usario realiza por ejemplo un clic o búsqueda. React necesita que mostrar, tipo una tarjeta, etc.
5. Renderizado de listas (.map + key): Este se usa en listamascota. Es el adecuado ya que es una forma eficiente donde renderizas un en este caso MascotaCard. Id es como una ke.
6. Renderizado condicionl: Este se usa en App.jsx cuando se muestra que no hay mascotas que coincidan. Es el adecuado por que no todo debe mostrarse siempre en la interfaz.
